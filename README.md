## Execution
From the project root:
1. Create a virtual env if you don't already have one:
```shell
    $ virtualenv venv
```
2. Activate and install MyHDL:
```shell
    $ source venv/bin/activate
    $ pip install myhdl
```
3. Run tests!
```shell
    $ python test/test_all_modules.py
```
